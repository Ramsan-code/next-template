# NearTailors Client

The frontend application for the NearTailors platform, built with:
- **Next.js 14** (App Router)
- **Tailwind CSS v4**
- **Shadcn/UI**
- **Zod** & **React Hook Form**

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000).

## Key Features
- **Authentication**: Login/Register with Zod validation.
- **Dashboard**: Protected route example.
- **Architecture**: Monorepo-ready structure.
